# Sleep Diary

A comprehensive web application for tracking sleep patterns and analyzing factors that impact sleep quality.

## Features

- **Week View**: See all 7 days in a table format for quick overview
- **Day View**: Focus on one day at a time with spacious form layout
- **Dark Mode**: Toggle between light and dark themes
- **16 Questions** tracking:
  - Sleep schedule (bedtime, wake time)
  - Sleep quality metrics
  - Sweet and caffeine intake
  - Screen time before bed
  - Pre-sleep activities
  - Stress levels
  - Sleep disturbances
- **Automatic Calculations**:
  - Sleep window
  - Time awake
  - Time asleep
  - Sleep efficiency percentage

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS3

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173/` to view the app.

### Build

```bash
npm run build
```

## Usage

1. Choose between **Week View** or **Day View** using the view switcher
2. Fill in your sleep data for each day
3. Track factors that may impact your sleep (caffeine, sweets, screen time, stress)
4. View automatic calculations at the bottom of the page
5. Toggle dark mode for nighttime use
